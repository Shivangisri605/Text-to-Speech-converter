# Text-to-Speech-converter

## Project Overview

This is a clean, modern, and highly functional web application that converts typed text into synthesized human speech using the browser's native **SpeechSynthesis API**. The project has been recently updated with a professional dark-mode theme for a sleek, modern user experience.

### Features

* **Speech Synthesis:** Converts text input into audio using the Web Speech API.
* **Input Validation:** Alerts the user if they try to convert an empty field.
* **Dynamic UI:** Button text changes dynamically to indicate when audio is playing.
* **Professional Design:** Utilizes a dark, minimalist color palette with responsive layout for a polished, modern look.

## ⚙️ Technologies Used

| **HTML5** | Project structure and semantic markup. |
| **CSS3** | Modern dark-mode styling with professional typography (Roboto). |
| **JavaScript** | Core logic using the built-in `window.speechSynthesis` API. |
